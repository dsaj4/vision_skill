# Package Workspaces

This directory is reserved for local runtime artifacts.

Workspaces are generated by evaluation and production commands, for example:

```text
package-workspaces/
  swot-analysis-workspace/
  golden-circle-workspace/
```

Generated workspace directories are intentionally not committed. They can contain:

- raw model responses
- transcripts
- grading and benchmark artifacts
- analysis outputs
- Kimi production-cycle task bundles
- temporary export files

The repository keeps only this README so the directory contract is visible in a clean checkout.
